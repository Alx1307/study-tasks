#Мини веб служба и конвейер обработки запросов

REST API для управления учебными задачами с демонстрацией конвейерной обработки запросов.

Использованный стек технологий: Node.js, Express, UUID.

Проект демонстрирует чистую архитектуру веб-приложения, где каждый запрос проходит через последовательность независимых обработчиков (middleware), каждый из которых отвечает за одну конкретную задачу:

1. **RequestIdMiddleware** — добавляет уникальный идентификатор к каждому запросу
2. **RequestLoggerMiddleware** — измеряет время выполнения и логирует успешные запросы
3. **ErrorHandlerMiddleware** — перехватывает исключения и формирует единый формат ошибок

## Установка и запуск

1. Клонировать репозиторий:
```
git clone https://github.com/Alx1307/study-tasks.git
cd study-tasks
```
2. Установить зависимости:
```
npm install
```
3. Запустить сервер:
```
cd src
node app.js
```

Сервер запустится на http://localhost:3000

## API Endpoints

GET	/api/tasks - Получить список всех задач

GET	/api/tasks/:id	- Получить задачу по ID

POST /api/tasks	- Создать новую задачу

## Правила валидации

title — обязательное поле, не может быть пустым

object — обязательное поле, не может быть пустым

priority — целое число 1, 2 или 3 (по умолчанию 3)

## Конвейер обработки запроса

Каждый запрос проходит через следующую последовательность:

Клиент → RequestId → RequestLogger → Маршрутизация → ErrorHandler → Ответ
